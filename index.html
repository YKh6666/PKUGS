PKUGS

Gaussian Splatting Compression Dataset for Large-Scale Scene Modeling with UAVs

A research-focused UAV dataset for large-scale scenes, built to support reproducible Gaussian Splatting compression workflows from raw captures to structured 3DGS assets.

Overview

Highlights

Large-Scale UAV Coverage

Scene captures are designed around wide-area outdoor reconstruction and long trajectories.

Compression-Oriented Packaging

Data releases align with 3DGS compression workflows and lightweight distribution targets.

Multi-Asset Release

Each sample may include raw videos, frame images, COLMAP artifacts, and Gaussian outputs.

Reproducible Splits

Versioned metadata helps maintain consistent splits and experiment reproducibility.

Research-First Licensing

The license is tailored for academic non-commercial usage with citation requirements.

JSON-Driven Frontend

Hero, samples, and changelog are generated from JSON files for fast project updates.

Data Contents

Raw Video

Original UAV capture sequences suitable for frame extraction and temporal analysis.

Extracted Images

Frame sets prepared for camera calibration and downstream reconstruction pipelines.

COLMAP Outputs

Sparse/dense reconstruction files for geometry and camera parameter initialization.

3DGS Outputs

Gaussian scene representations (for example, checkpoints and PLY exports).
